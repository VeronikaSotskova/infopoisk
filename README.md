# Как запустить

1. Установить [Python3](https://python-scripts.com/install-python)
2. Выполнить команду `pip install -r requirements.txt`

# Задание 1:
### Запуск:
1. Перейти в папку task_1
2. Выполнить команду `python crawler.py`
